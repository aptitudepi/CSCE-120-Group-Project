# Changelog

All notable changes to the Hyperlocal Weather project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive Makefile build system
- GitHub Actions CI/CD workflows
- Automated release creation
- Code coverage reporting
- Code quality checks (formatting, static analysis)
- Pull request templates
- Issue templates

### Changed
- Enhanced build system with multiple build types
- Improved test coverage
- Better error handling

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Hyperlocal weather forecasting
- Multi-source data aggregation (NWS, Pirate Weather)
- Precipitation nowcasting (0-90 minute forecasts)
- Geofenced weather alerts
- Offline caching with LRU cache
- SQLite database for persistence
- QML-based user interface
- Performance monitoring
- Comprehensive test suite

### Features
- Weather forecast display
- Location management
- Alert management
- Settings page
- Performance metrics dashboard
- Multi-service aggregation
- Nowcasting engine
